- Port Hearthbot core card/deck logic from Nonebot2 to AstrBot.
- Add AstrBot commands: `/card`, `/tags`, `/ori`, `/deck`.
- Add auto-detection for `AAE...` deck codes.
- Add compatibility guard for missing Battlegrounds pair card IDs in latest Hearthstone data.
- Fallback gracefully when HSReplay archetype data is unavailable.
